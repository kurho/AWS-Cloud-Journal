# Portfolio Cloud Journal App

A simple cloud-based journal application built using AWS core services.

## Architecture

User → EC2 Flask App → RDS MySQL  
Static assets → S3

## AWS Services Used

- Amazon EC2
- Amazon RDS
- Amazon S3
- Security Groups
- VPC concepts

## Future Improvements

- Add Application Load Balancer
- Add Auto Scaling Group
- Place EC2 and RDS in private subnets
- Add IAM role-based access
